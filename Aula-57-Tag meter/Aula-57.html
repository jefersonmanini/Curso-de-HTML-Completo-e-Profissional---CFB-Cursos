Tag meter

O elemento HTML meter (meter) pode representar um valor escalar dentro de um intervalo conhecido ou um valor fracionário.

Os textos dentro da tag meter só serão renderizados se o navegador não conseguir renderizar a TAG

Nota de uso: A não ser que o atributo value esteja entre 0 e 1(inclusive), o atributo min e o atributo max devem definir o intervalo de modo que o valor do atributo value esteja dentro dele.
