Tag base

O que faz a tag base? O elemento HTML Base () especifica o endereço (URL) utilizada por todos os endereços relativos contidos dentro de um documento.

Não é usado mais de uma tag dentro da página e é a prieira tag a ser usada.

Ela é usada dentro do head
