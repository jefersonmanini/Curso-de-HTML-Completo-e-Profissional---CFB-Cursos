Tag quote

O elemento HTML indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo; para citações maiores use o elemento blockquote.

Atriburos

Atributos Este elemento inclui os atributos globais. cite O valor deste atributo é uma URL que designa uma fonte ou mensagem para a informação citada. A intenção deste atributo é indicar a informação e explicar o contexto ou referência para a citação.

Exemplo 1

According to Mozilla's website, Firefox 1.0 was released in 2004 and became a big success.
