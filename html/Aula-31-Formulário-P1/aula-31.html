Formulário parte 1

A tag form cria um bloco para que tags de formulário e seus dados sejam agrupados e encaminhados para outro documento, em conjunto. Há muitas formas de se construir formulários em documentos HTML, e não é mais necessário que as tags de formulário como input ou select estejam contidas dentro de uma tag form

Os inputs submit, reset e button já vem com um texto pré-determinado, caso queira mudar o text é necessário usar a propriedade value.
