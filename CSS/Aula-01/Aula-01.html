O que seria o css?

Na programação existem vários tipos de lingugens e ferramentas que ajudam a desenvolver uma aplicação. Para o desenvolvimeto de aplicações webs nós temos o html, que já está na quinta versão, o css, versão 3 o javascript. Cada um tem uma função que ajuda no desenvolvimento web .

Enquanto o HTML tem como objetivo criar a estrutura principal do documento, a função do CSS é justamente otimizar o aspecto visual das páginas, de uma maneira mais organizada e menos complexa na hora realizar manutenção.

Para exemplificar o uso de cada um e como eles dão vida as aplicações, o html é a estrututa, ele que deixa o site de pé, podemos comparar como se fosse a alvenaria de uma casa sem o acabamento sem nada. O css é o acabamento, com as janelas personalizadas, tinta, massa, piso laminado e assim vai. javascript são as tecnologias como televisão, rádio, fogão, geladeira qualquer coisa que aumente a dinâmica da casa.

Usa de css

Existem três maneiras de inserir css

  1. Incorporado
  2. Externo
  3. interno

Interno

é usada uma tag dentro do head e tudo será feito dentro de uma tag na página html. (()). Está não é a forma mais recomendada pois, dependendo do tamanho do css irá deixar a página muito grande se desorganizada.

Externo

Neste é criado um arquivo css somente para o uso do html, pod criar um css para várias páginas ou um por página. Dependendo do tamanho do projeto é recomendável que seja um css para cada página

inline

Neste caso, cada tag irá ser estilizada individualmente.

Hierarquia sobre as maneiras de inserir o css

Como há três maneiras de inserir o css vem algumas dúvidas, será que as três maneiras se inseridas juntas irão funcionar da mesma forma casdo seja usada na mesma tag? Na verdade não, há uma hierarquia no uso do css, sendo o primeiro a ser usado o css interno ou inline. Isso acontece pois ele está sendo usado diretamente na tag. A segunda prioridade é o incorporado, o que é usado com a tag style, como ele também está na página fica muito mais fácil e rápido de ler já que não precisa ir em outros arquivos . E por último temos o CSS Externo, onde toda a stilização é feita em um arquivo a parte.
