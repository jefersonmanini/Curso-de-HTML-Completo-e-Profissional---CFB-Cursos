Formatação de Textos

Primeira propriedade é color. Ela muda a cor do elemento selecionado com o valor color.

Segundo elemento nós temos o text-align que configura o alinhamento do texto

propriedades do text-align

  • text-align: left - alinha a esquerda
  • text-align: right - alinha no centro
  • text-align: center - alinha no centro
  • text-align: justify - justifica

Segundo elemento nós temos o text-decoration remove o sublinhado padrão. A propriedade text-decoration do CSS é usada para definir a formatação de underline, overline, line-through ou blink. As decorações underline e overline são posicionadas abaixo e acima do texto (respectivamente), e line-through cortando-o.

A propriedade de CSS text-transform especifica como capitalizar um texto de um elemento. Pode ser usado para que o texto apareça com todas as letras maiúsculas ou todas minúsculas, ou com cada palavra em maiúscula.

            text-transform: capitalize;
            text-transform: uppercase;
            text-transform: lowercase;
            text-transform: none;
            text-transform: full-width;

Caso queira fazer paragráfo em um texto o elemento a ser usado é o text-indent. A identação só acontece na primeira linha de cada bloco, não afetando as restantes.

letter-spacing configura os espaço que há entre as letras em um texto. Se usarmos valores positivos os espaços irão aumentar, caso o valor usado seja negativo os espaços irão diminuir e as letras irão ficar mais próximas uma das outras.

line-height configura a altura da linha se for usado apenas números que dizer que a distância entre as linhas será calculada em cima do tamanho das linhas. Porém, pode ser usado as medias padrão como px, em ou %

Direction é a propriedade que inverte as letras assim como o html possui uma tag que faz a mesma coisa.
