Transformações 2D

As transformações 2d realizão o movimento entre eixos X e Y e no mesmo eixo seja deslocamento ou toação do elemento selecionado.

A propriedade CSS transform permite modificar o espaço coordenado do modelo de formatação CSS. Usando-a, elementos podem ser traduzidos, rotacionados, ter seu tamanho ajustado e inclinados de acordo com os valores definidos.

Translate

Em translate há uma movimentação da figura. O primeiro valor especiicado é no eixo X o segundo no eixo Y. Se ty não for especificado o valor presumido será 0

translate

Em rorate o elemento é rotacionado no seu próprio eixo e é usado graus para dizer o quanto ele vai rotacionar.
