outline

A propriedade CSS outline é uma propriedade abreviada para configurar uma ou mais das propriedades de contorno outline-style (en-US), outline-width (en-US) e outline-color (en-US) em uma única declaração. Na maioria dos casos o uso abreviado é preferível e mais conveniente.

Contornos se diferenciam de bordas das seguintes maneiras:

  • Contornos não ocupam espaços, eles são desenhados acima do conteúdo.
  • Contornos podem não ser retangulares. Eles são retangulares no Gecko/Firefox. Mas por exemplo, Opera desenha uma forma não retangular em torno de uma construção como este:

Sintexe

/* largura | estilo | cor */

outline: 1px solid white;

É importante notar que na div 1 há dois elementos que se destaca uma bordar preta e o que parece ser uma borda verde. Ao manipularmos a bordar preto percebemos que a distância entre todos os elementos aumentam ou se diminuirmos a distância entre os elementos diminui, poré isso não acontece com o elemento verde que seria o outline, a propriedade dele é justamente espaços poder varias de tamanho, porém sem alterar a distância entre os elementos.
