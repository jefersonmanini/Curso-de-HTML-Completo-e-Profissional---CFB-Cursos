Cores RGB e transparência

A função rgb() é uma função embutida em CSS que é usada para definir as cores usando o modelo Red Green Blue (RGB).

Sintaxe:

rgb( red, green, blue )

Parâmetros: esta função aceita três parâmetros conforme mencionado acima e descrito abaixo:

  • vermelho: este parâmetro é usado para definir a intensidade da cor vermelha. É um valor inteiro que varia entre 0 e 255 ou como um valor percentual entre 0% e 100%.
  • verde: este parâmetro é usado para definir a intensidade da cor verde. É um valor inteiro que varia entre 0 e 255 ou como um valor percentual entre 0% e 100%.
  • azul: este parâmetro é usado para definir a intensidade da cor azul. É um valor inteiro que varia entre 0 e 255 ou como um valor percentual entre 0% e 100%.

No css também é possível fazer com que a cor possa ser transparante. Após a sintaxe normal do RGB pode-se colocar o A que significa alpha. ficando como sintaxe rgba( red, green, blue, 0.6 )
