Propriedade order

A propriedade CSS order define a ordem para colocar um item em um contêiner flexível ou de grade. Os itens em um contêiner são classificados por valor de ordem crescente e, em seguida, pela ordem do código-fonte.

O uso da propriedade order criará uma desconexão entre a apresentação visual do conteúdo e a ordem do DOM. Isso afetará negativamente os usuários com baixa visão navegando com a ajuda de tecnologia assistiva, como um leitor de tela. Se a ordem visual (css) for importante, os usuários de leitores de tela não terão acesso à ordem de leitura correta.

flex-grow

A propriedade CSS flex-grow define o fator de crescimento flexível do tamanho principal de um item flexível.

Description

Esta propriedade especifica quanto espaço restante no contêiner flexível deve ser atribuído ao item (o fator de crescimento flexível).

Flex shrink

A propriedade CSS flex-shrink define o fator de redução flexível de um item flexível. Se o tamanho de todos os itens flexíveis for maior que o contêiner flexível, os itens encolherão para caber de acordo com o flex-shrink.

Em uso, flex-shrink é usado junto com as outras propriedades flex flex-grow e flex-basis, e normalmente definido usando a abreviação flex.
