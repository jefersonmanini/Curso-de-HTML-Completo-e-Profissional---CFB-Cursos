Formatações de tabelas

As bordas ao ser colocadas na tabela precisam ser colocadas não somente na tag table como todas outras tag, caso contrário ficará somente uma linha externa.

Metapropriedades

Em listas também é possível usar várias propriedades de uma vez.

Tamanho da tabela e das células.

Durante a formatação pode colocar várias propriedades e uma delas é width = largura, porém ao fazer isso as células não ficaram com tamanho iguais, mas a célular que tiver mais caracteres terá o maior tamanaho. Para uqe as células tenho o mesmo tamanho e fiquem bem distribuídas é necessário usar

Podemos controlar a altura de cada célula com height, porém ao fazer isso naturalmente o conteúdo irá ficar centralizado, mas podemos controlar isso e colocar onde quisermos com a propriedade vertical-align. É importante dizer que essa propriedade vertical-align só funciona com tabelas, caso alguém queira usar em uma div por exemplo é necessária transformá-la em tabela com a propriedade table-cell. Agora para alinha na horizontal usamos o text-align.

Desaparecendo com as linhas duplas nas tabelas.

Por padrão a tabela já vem com a propriedade "border-collapse: separeted", mas desaparecer com os espaços entre as células o ideal é usar a propriedade "border-collapse: collapse;"

Efeito listrado

Muitas vezes nos deparamos com tabelas enormes e precisamos tornar a identificação mais fácil, para isso precisamos diferencias a cor das linhas, colocar o efeito zebra. para isso podemos usar a pseudoclasse :nth-child(), ela seleciona elementos com base em suas posições em um grupo de elementos irmãos.

No exemplo da tabela foi usado o valor even, este valor representa os elementos cuja posição numérica em uma série de irmãos seja par: 2, 4, 6, etc. Existe também outro valor o odd, este valor representa elementos cuja posição numérica em uma série de irmãos seja ímpar: 1, 3, 5, etc.
