Uso de seletores

Os Seletores definem quais elementos um conjunto de regras CSS se aplica. Há três tipos básicos de seleção, tag, classe e id

Seleção por tag

Este seletor básico escolhe todos os elementos que correspondem ao nome fornecido.

  • nome-da-tag{ aqui dentro coloca a estilização desejada.}

Seleção por classe

Este seletor básico escolhe elementos baseados no valor de seu atributo classe. Quando for usar a classe no css primeiro será colocado um ponto . e depois a classe que foi usada no html para fazer a estilização. Pode existir vários elementos com a mesma classe

  • .nome-da-classe{ aqui dentro coloca a estilização desejada.}

Seleção por id

Este seletor básico escolhe nós baseados no valor do atributo id. Deve existir apenas um elemento com o mesmo ID no mesmo documento. para selecionar o id no css usamos # e depois o id da tag.

Exemplo: #toc irá corresponder ao elemento que possuir o id=toc (definido por um atributo id="toc", ou similar).
  • #nome-do-id{ aqui dentro coloca a estilização desejada.}

Elementos combinados

O combinador + seleciona os nós que seguem imediatamente o elemento especificado anteriormente.

Sintaxe: A + B

Exemplo: ul + li irá corresponder a qualquer elemento li que segue imediatamente após um elemento ul.
