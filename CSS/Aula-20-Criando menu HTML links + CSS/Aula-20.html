Criando menu HTML links + CSS

Para a criação de um menu o primeiro elemento necessário é o header, que seria o cabeçalho.

Segundo elemento é uma div, ela tem a função de delimitar a distância entre os dois lados.

Após isso será necessário colocar dentro da div a quantidade de div necessária de acordo com os seus elementos. Uma dessas divs será um menu e aí que iremos colocar a tag nav

Dentro da tag nav irá ter a quantidade de links necessários, ou seja a quantidades páginas necessáraias para o desenvolvimento do projeto.
