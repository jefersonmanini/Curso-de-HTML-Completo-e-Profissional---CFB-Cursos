Formatações para listas

As listas podem ser de dois tipos, ordenadas e desordenadas. Porém, mesmo elas já possuindo uma especificação, ainda sim ela podem ser formatdas diferente disso.

Em atributos que são dois, podemos alterar compact e o typo, este define o formado dos marcadores que irão aparecer. Em CSS podemos ir em "list-style-type:" e escolher o tipo de marcador.

Marcadores externos

Também é possível escolher uma imagem para ser o marcador das listas.

É interessante que quando for intopduzir imagens como marcadores elas sejam png com fundo branco para que não fiquem em contrastes.

Apesar do marcador fazer parte do todo da lista, na hora de formatar tanto o texto quanto o marcador são formatados diferentes. Sendo assim, é interessante qua quando for formatar o texto, a propriedade "list-style-position: inside;" seja usada.
