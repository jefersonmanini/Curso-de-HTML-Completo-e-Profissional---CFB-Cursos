Tag sup e sub

O sup tag define texto sobrescrito. texto sobrescrito parece metade de um caráter acima da linha normal, e às vezes é processado em uma fonte menor. Superscript texto pode ser utilizado para notas de rodapé, como WWW [1]. Dica: Use o sub tag para definir o texto subscrito.

Atributos

Este elemento inclui apenas os atributos globais.
